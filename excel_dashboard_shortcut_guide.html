ELIGIBLE / NOT ELIGIBLE — all status columns
3 min
1
Select columns AH through AM (click AH header, Shift-click AM header)
→ Alt H L N → New Rule → "Format only cells that contain"
→ Specific Text → containing → type ELIGIBLE
→ Format: Fill DCFCE7 · Font 15803D Bold → OK
2
New Rule → Specific Text → containing → NOT ELIGIBLE
→ Format: Fill FFE4E6 · Font BE123C Bold → OK
Make the NOT ELIGIBLE rule higher priority (move it up in the rules list with the ▲ button) so it overrides the ELIGIBLE rule when text says "NOT ELIGIBLE".
